# 🌦️ Weather App

A simple and responsive **Weather Application** built using **HTML, CSS, and JavaScript** that allows users to check real-time weather conditions for any city around the world. The application fetches live weather data from a weather API and displays key information such as temperature, weather conditions, humidity, and wind speed in a clean and user-friendly interface.

---

## 📌 Project Overview

The purpose of this project is to demonstrate how **frontend technologies** can interact with external APIs to display dynamic data. By entering a city name, users can instantly view the current weather details. The app updates the UI dynamically based on the data received from the API.

This project is beginner-friendly and focuses on **DOM manipulation, API integration, and responsive design**.

---

## ✨ Features

* 🔍 Search weather by city name
* 🌡️ Displays current temperature
* ☁️ Shows weather condition (cloudy, sunny, rain, etc.)
* 💧 Humidity information
* 🌬️ Wind speed details
* 📱 Responsive design for mobile and desktop
* ⚡ Fast and lightweight interface

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and responsive layout
* **JavaScript (ES6)** – Fetching API data and updating the UI
* **Weather API** – Provides real-time weather information

## ⚙️ How It Works

1. The user enters a **city name** in the search field.
2. JavaScript sends a **request to the weather API**.
3. The API returns weather data in **JSON format**.
4. The app extracts relevant information such as temperature and weather description.
5. The UI updates dynamically to display the results.

## 🚀 Getting Started

Follow these steps to run the project locally:

1. Clone the repository

```
git clone https://github.com/your-username/weather-app.git
```

2. Navigate to the project folder

```

```

3. Open `index.html` in your browser.
